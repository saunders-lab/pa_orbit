This github repository accompanies the manuscript and provides code notebooks that process and analyze raw data, turning it into preliminary figures in R. There are three main folders that are logically connected:

`/data -> /code -> /figures`

These code notebooks can be easily viewed as html documents. Links to these documents are listed below and are also available at the website version of this repository: <https://saunders-lab.github.io/pa_orbit/>
